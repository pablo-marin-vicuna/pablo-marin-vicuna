# Pablo Marín Vicuña

R&D AI Manager at Banco Falabella Chile. CTO & co-founder at [Skinology](https://skinology.cl) — a Chilean D2C skincare startup with an AI-driven formulation engine.

20+ years leading data science teams across financial services, airlines, and telecoms in Chile.
CMU MITS graduate (2024).

---

## What I build

- **Voice agents** — LLM-powered outbound calling systems (ElevenLabs + Twilio/Telnyx + Genesys) for financial services
- **AI product backends** — GCP-native, Cloud Run + Cloud SQL, FastAPI, RAG pipelines
- **Formulation AI** — rule-based engine for personalized skincare, where explainability matters more than black-box accuracy

## How I think about AI

I prefer explicit rules and traceable decisions over opaque models when the stakes are real.
I care about moving things to production, not just building prototypes.
